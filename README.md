
# 📄 DocuSense AI – Document Intelligence System

DocuSense AI is a full-stack web application that extracts and processes text from documents (PDFs and images) using Microsoft Azure Document Intelligence.  

It provides a modern, interactive interface with real-time OCR processing and smart output features.

---

## 🚀 Live Demo

🔗 https://docu-sense-ai--divyanshshukl38.replit.app/

---

## 💡 Features

- 📄 Extract text from PDFs and images using AI  
- ⚡ Real-time document processing  
- 🔐 Bring your own Azure credentials (endpoint & key)  
- 📋 Copy extracted text  
- ⬇️ Download extracted results  
- 🔊 Text-to-Speech functionality  
- 🎨 Premium UI with animations and modern design  

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask)  
- **Frontend:** HTML, CSS, JavaScript  
- **Cloud AI:** Microsoft Azure Document Intelligence  
- **Deployment:** Replit
